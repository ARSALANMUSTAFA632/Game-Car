🏎️ Nitro Racer Pro: Ultimate Edition (PWA)
Nitro Racer Pro is a high-speed, modern mobile web game developed using HTML5, CSS3, and JavaScript. More than just a website, it is a full Progressive Web App (PWA) that, once installed, functions seamlessly like a native Android application. 

✨ New & Key Features

🏎️ Car Unlock & Garage System:** اب آپ گیم میں جمع کیے گئے سکوں سے نئی گاڑیاں خرید سکتے ہیں۔ ہر گاڑی کی اپنی قیمت اور اسٹائل ہے۔

🎵 Procedural Sound FX:** High-fidelity audio for engine starts, jumping, coin collection, and crashes—all without extra assets!

💾 Persistent Storage:** آپ کی خریدی ہوئی گاڑیاں، اپ گریڈز، سکے اور ہائی اسکور LocalStorage میں ہمیشہ محفوظ رہتے ہیں۔

🛒 Economy System (Shop):** Use your hard-earned coins to upgrade your **Top Speed** and Shield Duration.

💾 Cloud-like Storage:** Your upgrades, coins, and high scores are persistent via LocalStorage.

🆙 Dynamic Leveling System: Progress through levels every 500 points! Experience increased speed and escalating challenges as you advance.

🎁 Daily Rewards: Get a boost with 50 free coins every 24 hours just for opening the game.

🚧 Obstacle Variety: Face new challenges with varying obstacle sizes (Tall & Long), requiring precise jump timing to overcome.

🕹️ Hybrid Controls: Optimized for all devices—tap anywhere on the screen for mobile play or use the Spacebar on laptops and PCs. 

🛡️ Advanced Power-ups:

Shield: Grants 7 seconds of total invulnerability against all obstacles.

Magnet: Automatically attracts nearby coins toward the vehicle.

💾 Auto Data Saving: Your high scores, levels, and coins are automatically saved to your device using the LocalStorage API.

🎨 Modern UI/UX: Features a sleek Glassmorphism design with smooth, high-performance animations.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Modern Flexbox & Animations).

Logic: JavaScript (ES6+) – Enhanced with Leveling and Reward algorithms.

Configuration: Web App Manifest (JSON) for PWA compliance.

Storage: LocalStorage API.

🚀 Installation Guide
Host the project files on your server or GitHub Pages.

Open the game link in your mobile browser (Chrome or Safari).

Navigate to the browser menu (three dots) and select "Add to Home Screen" or "Install App".

The game is now available in your app drawer!
